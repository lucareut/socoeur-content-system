# Socoeur — Social Media & Content Plan (World Cup 2026 Launch)

## Context

Brand: Socoeur  
Positioning: Cultural fashion brand referencing football  
Tagline: Different roots, same colors

Key objective:  
Launch the brand and first jersey capsule leading up to the World Cup 2026.

Limited stock: 300–500 jerseys.

Target audience:
Urban Switzerland (Geneva, Lausanne, Fribourg, Zurich, Basel, Bern)  
Age: 18–50  
Lifestyle: street culture, design, football affinity but not hardcore fans.

Visual direction:
Architecture + fashion + minimalism.

# Production Workflow

Batch production approach.

One shooting session should produce assets for multiple posts.

Typical session:

25–40 minutes shooting  
45 minutes editing

Each session should generate content for 4–6 posts.
